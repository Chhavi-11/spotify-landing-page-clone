# Spotify Landing Page Clone

A responsive and visually polished clone of the Spotify landing page built using HTML5 and CSS3.
This project focuses on clean UI design, mobile responsiveness, proper layout structuring, and modern CSS styling practices.

# Features

Fully responsive landing page

Pixel-perfect sections inspired by Spotify

Clean and modern UI layout

Smooth hover effects

Organized and readable code

Mobile-first approach

Pure HTML + CSS

# Visual Styling

Custom color palette inspired by Spotify

Background gradients

Border-radius for rounded components

Box-shadow for depth

Opacity adjustments

Hover transitions for interactive UI elements
